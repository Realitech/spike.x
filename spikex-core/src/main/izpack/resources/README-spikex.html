Welcome to Spike.x!

This will install and setup Spike.x [http://www.spikex.io].
Reactive event monitoring and analysis built on top of Vert.x. Bridging common monitoring agents with modern time series databases and dashboards.
Spike.x provides components for resource monitoring, for data filtering and streaming, for sending of notifications, for storing of events in various backends and for many other use cases.
Spike.x can be installed on many platforms that support JDK 1.8 or newer. The following platforms are currently supported:
  • Windows 64-bit (Windows 7, Windows 2003 Server, etc.)
  • Linux 64-bit (Ubuntu, CentOs, RedHat, etc.)
  • Apple OS X 64-bit (Mavericks and newer)
  • FreeBSD 64-bit (FreeBSD 10 and newer)
Spike.x can be used out-of-the-box for the following use cases:
  • Passive monitoring of operating system and application level metrics
  • Receiving metrics from NRDP agents (NSClient++) or collectd
  • Notifying by email when thresholds have been reached
  • Storing the metrics and events in InfluxDB or Elasticsearch
  • Presenting the collected metrics with Grafana or Kibana
